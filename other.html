ZFC 以外の集合論

\(ZFC\)以外の集合論の一つに\(NBG\)(Von Neumann–Bernays–Gödel set theory)があります． これは\(ZFC\)では非公式に扱っていたクラスを公式に扱うようにしたことで，公理図式が一つの公理にまとめられるようになります． またこれは\(ZFC\)の保守拡大であり集合に限定すれば同じ定理が成り立ちます．

\(NBG\)と近い集合論として\(MK\)(Morse-Kelley set theory)があります． これは内包性公理図式の式の変数の範囲をクラス全体に拡大したものです． これにより\(NBG\)よりも強い理論となったが公理図式が必要になりました．

\(ZFC\)とは別の方法で内包性公理図式を制限するものもあります． その一つが\(NF\)(New Foundations)です． これでは内包公理図式の式を制限しています． 型理論のように集合が階層化されている式でのみ内包性公理図式が成り立ちます． また，選択公理は成り立たず，万有集合の存在を示せます．
